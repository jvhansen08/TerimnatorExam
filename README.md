# TerimnatorExam

# Setup

Create the virtual environment

```bash
python3 -m venv .venv
```

Activate the virtual environment

MacOS

```bash
source .venv/bin/activate
```

Windows

```bash
.venv\Scripts\activate.bat
```

Install the requirements

```bash
pip install -r requirements.txt
```

Deactivate the virtual environment

```bash
deactivate
```
